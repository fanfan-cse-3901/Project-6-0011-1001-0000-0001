### Setup

#### Step 1: Install Dependencies

```
# Debian/Ubuntu
$ gem install bundler
$ sudo apt install g++ libxrandr-dev libfox-1.6-dev
```

#### Step 2: Install Gem Packages

```
# Debian/Ubuntu
$ bundle install
```

#### Step 3: Compile the code

```
# Run rails server
$ rails server
# Type localhost:3000 in the browser and go.
```
